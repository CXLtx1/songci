# AI宋词相似性分析数据说明文档

## 一、项目概述

本项目旨在对大语言模型(LLM)生成的宋词进行相似性分析，评估模型生成内容的同质化程度，识别潜在的"高级伪原创"问题（语义相似但字面不同）。

## 三、分析方法说明

### 3.1 相似度计算方法

#### 语义相似度（BERT Embedding）

**模型**：BERT-CCPoem（清华大学古诗词专用BERT模型）

**计算流程**：
1. 将文本输入BERT模型获取embedding向量
2. 计算cosine相似度：`sim = cos(v1, v2) = (v1·v2) / (|v1|×|v2|)`
3. 输出范围：0-1，越高表示语义越相似

**特点**：
- 衡量意境、情感、主题的相似性
- 不受字面差异影响
- 能识别"换词不换意"的伪原创

#### 字面相似度（difflib）

**算法**：SequenceMatcher（最长公共子序列LCS）

**计算公式**：`sim = 2 × LCS长度 / (len1 + len2)`

**特点**：
- 衡量字词、句式的重复程度
- 考虑顺序和位置
- 直接反映抄袭程度

### 3.2 分析粒度

| 分析粒度 | 说明 | 应用场景 |
|---------|------|---------|
| **词级别** | 整首词作为一个单元 | 评估整体风格相似性 |
| **句级别** | 每个句子作为单元 | 发现局部重复问题 |

## 五、关键指标解读

### 5.1 综合评分

**公式**：`综合评分 = 0.55 × 语义P50 + 0.45 × 语义P90`

**解读**：
| 评分范围 | 同质化程度 |
|---------|-----------|
| < 0.35 | 多样性良好 |
| 0.35-0.40 | 轻度同质化 |
| 0.40-0.45 | 中度同质化 |
| > 0.45 | 重度同质化 |

### 5.2 伪原创指数

**公式**：`伪原创指数 = 语义相似度 - 字面相似度`

**解读**：
| 指数范围 | 问题类型 |
|---------|---------|
| < 0.2 | 正常相似或直接重复 |
| 0.2-0.4 | 轻度伪原创 |
| 0.4-0.6 | 中度伪原创 |
| > 0.6 | 重度伪原创（换词不换意） |

### 5.3 分位数指标

| 分位数 | 含义 |
|--------|------|
| P25 | 25%的句对低于此值 |
| P50（中位数） | 一半句对的相似度水平 |
| P75 | 75%的句对低于此值 |
| P90 | 10%的句对高于此值（严重重复） |
| P95 | 5%的句对高于此值（极端重复） |

## 六、可视化图表说明

### 6.1 箱线图（Boxplot）

**用途**：展示各模型相似度的整体分布

**解读要点**：
- 箱体位置越高 → 同质化越严重
- 箱体越窄 → 相似度集中，风格一致
- 箱体越宽 → 多样性好

### 6.2 热力图（Heatmap）

**用途**：快速定位问题严重的模型+词牌组合

**解读要点**：
- 颜色越深（红） → 同质化严重
- 颜色越浅（蓝） → 多样性好

### 6.3 KDE密度曲线

**用途**：精细展示相似度的概率密度分布

**解读要点**：
- 峰值位置靠右 → 大部分句对高相似
- 峰值位置靠左 → 大部分句对低相似
- 多峰分布 → 存在多个相似模式

### 6.4 雷达图（Radar）

**用途**：多维度综合评分对比

**解读要点**：
- 面积越大 → 同质化越严重
- 形状偏向某维度 → 该维度问题突出

### 6.5 网络图（Graph）

**用途**：可视化高相似句对/词对的关系网络

**解读要点**：
- 节点大小 → 连接数（相似句对数）
- 连线粗细 → 相似度高低
- 聚类现象 → 存在相似群体

## 七、数据使用建议

### 7.1 分析流程建议

1. **整体评估**：查看模型级统计表，了解各模型整体同质化程度
2. **词牌定位**：查看热力图，定位问题严重的词牌
3. **深入分析**：查看词牌级统计表，分析具体词牌的相似性特征
4. **案例研究**：查看Hidden Connections CSV，分析具体的高相似句对/词对
5. **可视化辅助**：结合网络图，直观理解相似关系

### 7.2 关键问题识别

| 问题类型 | 识别方法 | 数据来源 |
|---------|---------|---------|
| 整体同质化 | 综合评分高、P50/P90高 | 模型级统计表 |
| 局部重复 | P90明显高于P50 | 模型级统计表 |
| 高级伪原创 | 伪原创比例高、差值P50高 | Hidden Connections表 |
| 词牌固化 | 词牌级综合评分高 | 词牌级统计表 |

## 八、技术细节

### 8.1 BERT-CCPoem模型

**来源**：清华大学自然语言处理实验室

**特点**：
- 专为古诗词设计的BERT模型
- 在大规模古诗词语料上预训练
- 对语义相似度计算更准确

**模型路径**：`analyze/BERT-CCPoem-Model/`

### 8.2 计算效率优化

| 优化方法 | 说明 |
|---------|------|
| 向量化计算 | 使用numpy批量处理索引 |
| 多进程并行 | 4核并行计算字面相似度 |
| 堆内存控制 | 只保留Top N高相似句对 |
| 流式处理 | 边计算边处理，避免内存爆炸 |
