# 结果

## 1. 不同创作约束下 AI 宋词内部同质化的总体变化

基于 [neo_model_summary.csv](/D:/文件/Projects/songci/result/neo-analysis/neo_model_summary.csv) 的严格主分析结果可见，AI 宋词的内部同质化程度会随创作约束增强而显著上升。在固定五词牌、无主题的 `b1` 条件下，主模型的 `mihi_full` 大致分布在 `0.439374` 到 `0.506810` 之间；当加入主题约束后，`b2` 与 `b3` 的 `mihi_full` 分别上升到 `0.464420-0.517539` 与 `0.518089-0.531132`；进一步固定为单一词牌 `沁园春` 后，`c1` 与 `c2` 的 `mihi_full` 则升至 `0.595676-0.642379`。这表明，词牌约束、主题约束与单词牌强约束会共同压缩模型输出空间，并推动作品在内部相互逼近。

在具体模型上，不同模型对约束的敏感性并不一致。以 `b1` 为例，`qwen3.6-plus` 的 `mihi_full` 为 `0.506810`，高于 `doubao-seed-2.0-pro` 的 `0.490574` 和 `deepseek-3.2-thinking` 的 `0.450301`；而在 `c1` 条件下，`doubao-seed-2.0-pro` 和 `deepseek-3.2-thinking` 的同质化程度则分别达到 `0.636859` 和 `0.630861`，明显高于 `qwen3.6-plus` 的 `0.595676`。这说明不同模型并不存在固定不变的“最同质化”排序，而是对不同类型的创作约束呈现出不同的响应模式。

## 2. 主题约束的差异性作用

条件差分结果表明，主题约束会进一步提高模型内部同质化，但不同主题带来的提升幅度并不相同。根据 [neo_condition_deltas.csv](/D:/文件/Projects/songci/result/neo-analysis/neo_condition_deltas.csv)，从 `b1` 到改革主题条件 `b2`，三家主模型的 `delta_mihi_full` 分别为 `+0.014119`、`+0.026965` 和 `+0.006895`；而从 `b1` 到玉兰主题条件 `b3`，对应增幅则为 `+0.072638`、`+0.040558` 和 `+0.011278`。可以看到，玉兰主题在三家主模型上均表现出更强的同质化诱发效应，尤其在 `deepseek-3.2-thinking` 上最为明显。

这一差异说明，主题并不是均匀地压缩表达空间，而是会根据其语义性质触发不同程度的模板收缩。与“改革”相比，“玉兰”更容易将模型推向相似的意象资源、相似的寄托方式以及相近的抒情路径，因此在整体上更容易形成高密度的内部同质化分布。

## 3. 同质化首先表现为整体语义收缩

进一步分解 `mihi_full` 的组成可以发现，主题约束带来的同质化提升首先表现为整词语义层的收缩，而不是句级模板的同步强化。在 `b1 -> b2` 的比较中，三家主模型的 `delta_whole_score` 分别为 `+0.073021`、`+0.085857` 和 `+0.069881`，而 `delta_sent_score` 则分别为 `-0.065387`、`-0.045697` 和 `-0.059620`。在 `b1 -> b3` 的比较中，这一趋势同样存在：`delta_whole_score` 依然整体为正，而 `delta_sent_score` 多数仍为负。

这表明，当主题被固定后，模型首先在“整首词表达的核心意义与情感方向”上趋同，即不同作品越来越倾向于围绕相同的主题意涵和情绪框架展开；相比之下，局部句法结构和具体表达方式在这一阶段尚未完全同步收缩。因此，主题约束带来的首要变化是整体语义同质化，而不是全面的句级复用。

## 4. 强结构 prompt 对句级模板复用的推进

当约束从“五词牌 + 固定主题”进一步推进到“单一词牌 + 固定主题 + 更强结构提示”时，句级模板复用的作用开始增强。以 `b3 -> c1` 为例，`doubao-seed-2.0-pro` 的 `delta_whole_score` 仅为 `+0.000348`，但其 `delta_sent_score` 却上升了 `+0.030238`，最终带动 `delta_mihi_full` 增加 `+0.009034`。相较之下，`deepseek-3.2-thinking` 和 `qwen3.6-plus` 在该对比中则未表现出同样的句级抬升。

这一结果提示，`type_a` 这类包含显式格律模板的强结构 prompt，不仅会约束作品的词牌框架，也可能进一步压缩局部句法和表达骨架，使部分模型在句级层面出现更明显的复用。换言之，当结构提示足够强时，句级模板同质化会从次要因素转化为推动整体同质化上升的重要来源。

## 5. AI 相对于真实宋词的超额同质化

与真实宋词基线相比，AI 模型在所有主要实验条件下都表现出明显的超额同质化。以 `excess_core_homogeneity` 为例，`b1` 条件下四个可比较模型的数值分别为 `+0.092521`、`+0.075996`、`+0.031228` 和 `+0.031132`；在 `b2` 条件下，这一数值进一步提升到 `+0.071425` 至 `+0.108029`；在 `c1` 和 `c2` 条件下，则达到 `+0.152833` 至 `+0.212229` 的区间。

这意味着，AI 生成宋词不仅在模型内部趋于相似，而且其相似性已经显著高于真实宋词在相同词牌范围内的自然创作分布。尤其在固定单一词牌和固定主题的强约束任务中，这种“超额同质化”表现得最为显著。

## 6. 自由生成条件下的探索性结果

在 `a1` 的自由生成条件下，模型结果仅作探索性观察，不纳入严格横向排名。尽管如此，该条件下仍可观察到较为明显的内部集中现象。例如，`qwen3-max` 的 `mihi_full` 为 `0.603720`，高于其他自由生成模型；结合其词牌分布可以看到，该模型在自由生成下高度集中于少数词牌，这说明自由条件并未自动保证输出分布的充分发散。

因此，`a1` 的结果更适合用于说明模型在弱约束下的默认收缩倾向，而不宜与 `b1/b2/b3/c1/c2` 的受控条件直接进行严格排名比较。

## 7. 主题类型与意象系统对同质化的补充解释

描述性统计结果进一步表明，AI 宋词的同质化并不只体现在语义相似度上，还体现在题材归类和意象资源的收缩上。数据库级 `category` 分布显示，`a1` 与 `b1` 中最常见的主题类型均为 `离别羁旅`，占比分别为 `34.33%` 和 `42.71%`；在改革主题条件 `b2` 中，作品主要集中于 `谈禅说理`（`41.08%`）与 `咏物寄托`（`39.65%`）；而在玉兰主题条件 `b3` 和 `c1` 中，`咏物寄托` 的占比分别达到 `92.05%` 和 `87.66%`。这说明同质化不仅是文本层面的相似，也表现为主题类型的高度集中。

意象统计进一步揭示了收缩发生的内容层机制。`b3` 与 `c1` 的高频意象集中在 `玉兰、寒窗、素萼、玉树、琼枝、琼英、清香、春寒` 等一组稳定意象上；`b2` 与 `c2` 则围绕 `樊笼、灵根、青衿、晨钟、暮鼓、宗门、铁律、剑气` 等意象形成较稳定的改革语义场。由此可见，许多同质化并不是整首词完全重复，而是模型能够调用的主题意象资源池不断变窄。

## 8. 数据库间意象重叠率的补充证据

数据库间意象重叠率进一步支撑了上述判断。根据 [imagery_overlap_between_dbs.csv](/D:/文件/Projects/songci/result/descriptive-stats/imagery_overlap_between_dbs.csv)，`b3` 与 `c1` 的 `distinct_imagery_jaccard` 为 `0.274739`，`top20_overlap_count` 为 `16`，共享核心意象包括 `玉兰、寒窗、春寒、春风、琼枝、琼英、素影、素萼、长街、青衫` 等；`b2` 与 `c2` 的 `distinct_imagery_jaccard` 为 `0.219302`，`top20_overlap_count` 为 `14`，共享核心意象包括 `樊笼、灵根、青衿、剑气、剑阁、劫火、宗门、晨钟、暮鼓、铁律` 等。

与之相比，无主题条件与主题条件之间的核心意象重叠明显更低。例如 `a1` 与 `c2` 的 `top20_overlap_count` 为 `0`，`b1` 与 `c2` 的 `top20_overlap_count` 也为 `0`。这说明主题不仅提高了文本相似度，还会系统性地重塑模型调用的核心意象资源库，使作品在内容层面呈现出更强的聚集和重复。

## 9. 小结

综合来看，本研究表明 AI 宋词的内部同质化会随着创作约束增强而持续上升，其中玉兰主题比改革主题更容易诱发模板化；这种同质化首先表现为整体语义收缩，随后在强结构 prompt 下进一步扩展到句级模板与意象系统层面。与此同时，AI 在主要实验条件下均表现出高于真实宋词基线的超额同质化。描述性统计进一步说明，这种同质化并非仅仅是“文本更像”，而是表现为词牌选择、主题归类和意象资源的多层收缩。
